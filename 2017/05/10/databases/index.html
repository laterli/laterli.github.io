数据库的命令符操作指令

只看上边可能会有些思维混乱下面我列一个树状图出来。

```
数据库 —增加 create database 数据库名，
—删除 drop database 数据库名，

表 —增加 create 表名(字段名 类型1 类型2···，字段名 类型1，类型2···)
—删除 drop table 表名

表中数据 —增加 insert into 表名(字段名，字段名···) value(新值，新值···)
上述操作可以在表头有id且有自动生成时自动生成id;还有一种方法
insert into 表名 values(新值，新值，新值···)
此操作不会自动添加id;
—删除 delete from 表名 where 条件；
—修改 update 表名 set 字段名=新值 where 条件；
—查看 select * from 表名 where 条件；
